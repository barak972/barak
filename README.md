# Room Booking App

אפליקציית הזמנת חדרי ישיבות עבור אצטדיון מרים.
מפותחת ב-React עם שמירת נתונים ב-Firebase Realtime Database.

## איך מריצים מקומית

```bash
npm install
npm start
```
